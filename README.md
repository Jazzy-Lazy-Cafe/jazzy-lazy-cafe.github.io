# Jazzy Lazy Cafe - Root Domain

This repository serves as the root landing page for `https://jazzy-lazy-cafe.github.io`.

It automatically redirects visitors to the main magazine site at `https://jazzy-lazy-cafe.github.io/ai-magazine/`.

## Setup Instructions

1. This repository must be named exactly: `jazzy-lazy-cafe.github.io`
2. Enable GitHub Pages:
   - Go to repository Settings â†’ Pages
   - Source: Deploy from a branch
   - Branch: `main` (or `master`)
   - Folder: `/ (root)`
   - Click Save

3. Wait 1-2 minutes for deployment

4. Visit `https://jazzy-lazy-cafe.github.io` to see the redirect page
